Linear Style Guide

A comprehensive design reference extracted from Linear's marketing site and application interface. Use as a foundation for building applications with Linear's distinctive aesthetic.

Linear's palette is built on a near-black foundation with carefully calibrated grays and a single purple-blue accent. The dark theme dominates, creating depth through subtle tonal shifts rather than heavy borders or shadows.

Typography

Linear uses Inter Variable exclusively, leveraging its optical sizing and custom font-feature-settings (cv02 cv03 cv04 cv11). The type scale is tight and controlled, with negative letter-spacing on headings giving them a premium, compact feel.

Design Principles

Core patterns that define Linear's visual identity and should be carried into your application.

Dark-first, depth through tone

No box-shadows anywhere. Depth is created purely through background color stepping: page → card → elevated → hover. Each step is a barely-perceptible lightening.

Compact & information-dense

Controls are small (30px height), text is 13px for UI labels, and padding is restrained. The interface prioritizes information density without feeling cramped.

Tight letter-spacing on headings

All headings above 24px use negative letter-spacing (from -0.288px to -1.408px). This tightening gives the typography its premium, designed feel.

Minimal accent color usage

The purple-blue accent (#828fff) appears only on interactive links and active toggles. The interface is almost entirely monochromatic, making the rare accent usage impactful.

Frosted glass navigation

The top navbar uses backdrop-filter: blur(20px) with a semi-transparent background, creating a subtle frosted effect as content scrolls behind it.

Weight 510 is the signature

Inter's 510 weight (semibold) is Linear's distinctive choice. It's used on hero headings, buttons, category labels, and links. Heavier than 500, lighter than 600, uniquely Linear.
